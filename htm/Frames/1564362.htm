Ótimo para overclocking

 

Cada módulo de memória Vengeance é construído a partir de DRAM ICs especialmente selecionados para potencial de desempenho, para que você possa explorar os limites do sistema de overclocking.

 

 

Se você é novo para overclocking, suporte completo para o perfil Intel Extreme Memory (Intel XMP) torna mais fácil. Adicionar Vengeance para o seu sistema XMP compatível e pode selecionar automaticamente o desempenho máximo possível overclocking, rápida e facilmente.
