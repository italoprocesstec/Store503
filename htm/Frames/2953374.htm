Dock Station Targus ACP70US-50 Usb 3.0 PCs, Macs, e Android

Aumentar a produtividade no local de trabalho com uma docking station que liga todos os periféricos essenciais de uma vez.

 

 

Visão Geral

 

Desfrute de um aumento na velocidade e mais poder, utilizando a mais recente tecnologia USB 3.0 para o seu workstation.The Targus USB 3.0 SuperSpeed ​​Docking Station dupla Vídeo oferece duas portas de vídeo que oferecem além de vídeo HD e gráficos até 2048 x 1152 (1x DVI-I, 1x HDMI) para dois monitores. Além disso, inclui um adaptador DVI para VGA, um adaptador HDMI para DVI, 2 portas USB 3.0 para o seu USB 3.0 periféricos e quatro portas USB 2.0 para todo o seu USB 2.0 itens.

 

A docking station também inclui uma porta Gigabit Ethernet, áudio In / Out portas para conectar alto-falantes e um slot para trava de segurança para ajudar a proteger a doca e evitar o roubo. Um adaptador 6A AC e cabo USB 3.0 dois pés também estão incluídos para ajudar a tornar mais fácil para conectar o dispositivo a uma estação de trabalho completa - incluindo dois monitores, um mouse, teclado, impressora e outros periféricos que você pode precisar para se conectar com, basta encaixar o laptop . A docking station também tem duas portas USB de alta potência, fornecendo energia extra e permanece acesa mesmo quando o laptop é desligado
