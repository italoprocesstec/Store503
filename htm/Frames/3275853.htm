Informações do produto

 

Windows Server 2012 Standard O Windows Server 2012 Standard é um sistema operacional projetado para aumentar a flexibilidade da infraestrutura de servidores e ajudar a poupar tempo e reduzir custos Desenvolvido para cargas de trabalho mais exigentes o Windows Server 2008 R2 Standard contém recursos inovadores como virtualização economia de energia e gerenciamento e ajuda os funcionários que trabalham remotamente acessarem os recursos da empresa com mais facilidade Cenários Alta disponibilidade evitando paradas no sistema e indisponibilidade no serviço da empresa Eficiência de TI com gerenciamento simplificado Ideal para atender às demandas de aplicações de missão crítica Criado para ambientes físicos ou minimamente virtualizados Limite de criação de até duas máquinas virtuais Necessita de CAL de usuário ou de dispositivo para acessar o servidor
