Armazenamento simples e instantâneo

 

 

 

Visão Geral

 

O disco portátil Seagate® Expansion é compacto e perfeito para ser transportado. Adicione mais espaço de armazenamento instantaneamente ao seu computador e leve arquivos grandes nas suas viagens.

 

A instalação é simples e direta, bastando conectar um cabo USB e pronto. O disco é alimentado pelo cabo USB, dispensando uma fonte de alimentação externa. Além disso, ele é reconhecido automaticamente pelo sistema operacional Windows®, então não há software para instalar nem nada para configurar. Salvar arquivos é fácil

 

— é só arrastar e soltar. Aproveite ao máximo as velocidades de transferência de dados rápidas da interface USB 3.0 conectando o disco a uma porta USB 3.0 SuperSpeed. A interface USB 3.0 é compatível retroativamente com USB 2.0, proporcionando maior flexibilidade
