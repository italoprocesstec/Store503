Atualize seu acesso à rede

 

 

Visão Geral

A série de switches N1500 oferece uma Gigabit Ethernet (GbE) solução de comutação de acesso à rede com eficiência de energia com uplinks de 10GbE integrados. A série de switches N1500 tem capacidades de alto desempenho e desempenho wire-speed, utilizando uma arquitetura non-blocking para lidar com cargas de tráfego inesperadas facilmente. Estes switches oferecem gerenciamento simples e escalabilidade através de um 40Gbps (full-duplex) arquitetura de empilhamento de alta disponibilidade que permite o gerenciamento de até quatro chaves a partir de um único endereço IP.

 

 

Fornecer energia limpa para dispositivos com suporte PoE + rede

A Rede Dell Série N1500 tem opções para até 48 Power-over-Ethernet Plus (PoE +) portos. Você pode conectar dispositivos com requisitos de alta potência (até 30,8 watts) sem executar um cabo de alimentação separado. PoE + é particularmente útil em edifícios mais antigos, onde a instalação de energia para vários locais, difundidas pode ser proibitivamente caro

 

 

Projetado para a eficiência

Construído para ajudar a manter os custos operacionais baixos, a série N1500 pode operar em temperaturas de até 113 ° F (45 ° C) para ajudar a reduzir os custos de resfriamento. Melhorar a sua eficiência global com estas grandes características:

- 80 Plus certificada eficiente fonte de alimentação integrada

- Ventiladores redundantes de refrigeração de velocidade variável

- Green Ethernet e camada física de baixa potência (PHYs) para ajudar a reduzir o poder aos portos inativos e links inativos

 

 

Simplificar o gerenciamento com recursos avançados

A Dell Networking Series N1500 oferece fácil implantação, a interoperabilidade excelente e uma curva de aprendizado nível para administradores de rede. E com Rede Dell OS 6, você mantém configurações consistentes em todos os produtos N-Series com recursos abrangentes:

- Um comando comum Line Interface (CLI) e interface gráfica do usuário (GUI) que utilizam uma linguagem de comando bem conhecido para ajudar os administradores de rede qualificados se tornar produtivo rapidamente

- Auto-configuração USB que ajuda os administradores de rede implantar rapidamente configurações espelhadas para inúmeros dispositivos, simplesmente inserindo uma chave USB

- Uma variedade de empresa de classe apresenta com comandos familiares e intuitivas de configuração e gerenciamento, incluindo os mais recentes protocolos de padrão aberto e tecnologias de integração inteligentes para ajudar a tornar as redes de vários fornecedores que operam sem problemas possível

 

 

Implante com confiança em qualquer escala

Com alto rendimento e capacidade de lidar com cargas de trabalho inesperados, os interruptores N1500 Series são soluções ideais para redes de rápido crescimento GbE que requerem a agregação de alta densidade com redundância sem costura e disponibilidade. Aproveite os seguintes recursos:

- As taxas de dados de até 176Gbps (full-duplex) e uma taxa de transmissão de até 164Mpps

- Quatro 10GbE SFP + para uplinks de alta velocidade e um empilhamento arquitetura até quatro unidades 40Gbps

- Interruptor acumula até 200 portas 1GbE que podem ser gerenciados a partir de uma única tela ou endereço IP
