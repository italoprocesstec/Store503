Seus servidores precisam de unidades de disco SAS de alto desempenho para aplicativos críticos para o negócio e voltados para armazenamento?

 

 

Visão Geral

 

As unidades de disco rígido HPE SAS para aplicações intermediárias e corporativas oferecem soluções de alto desempenho para hospedar aplicativos baseados em transações altas. Elas estão disponíveis no HPE Smart Carrier, transportador hot-plug padrão HPE e transportador sem hot-plug HPE.

 

As unidades de disco rígido HPE SAS estão disponíveis nos formatos pequeno, de 2,5 polegadas (SFF), e grande, de 3,5 polegadas (LFF), e vêm em várias capacidades para atender a uma ampla gama de requisitos, dando suporte a aplicações corporativas e intermediárias. A interface SAS foi especialmente projetada para oferecer alta disponibilidade onde o armazenamento de dados de nível empresarial, o desempenho, a confiabilidade e a integridade dos dados são cruciais.

Desempenho e confiabilidade para ambientes com vários aplicativos

As unidades de disco rígido HPE SAS oferecem alta confiabilidade para aplicativos críticos para o negócio.

Interface SAS robusta (porta dupla, full duplex) para aumentar o desempenho.

HPE Smart Carrier para servidores HPE ProLiant (Gen8 e Gen9 são suportados). Mantém a compatibilidade regressiva com transportadores hot-plug padrão existentes, a fim de fornecer suporte para servidores HPE ProLiant G7 e servidores e compartimentos de armazenamento de servidor anteriores.

Elas são ideais para processamento de transações, aplicativos de banco de dados, análises, computação de alto desempenho e aplicações de negócio.

A unidade de formato avançada 512e é ideal para aplicativos e sistemas operacionais 4KiB alinhados. Eles também suportam aplicativos 512K nativos legados.
