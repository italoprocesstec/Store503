Permite que um único usuário acesse o servidor Microsoft a partir de diversos dispositivos

 

 

Visão geral

 

CAL Windows Server CAL 2012 Brazilian 1pk DSP OEI 5 Clt User CAL Quando uma empresa compra qualquer versão do Windows Server, ela adquire a licença para instalar o sistema operacional em um servidor, porém para que os computadores clientes (desktops) possam acessar os aplicativos deste servidor é necessário que a empresa adquira também as chamadas Client Access Licenses (CALs). As versões Standard e Datacenter continuam exigindo CALs que devem ser atribuídas a cada usuário ou dispositivo que acesse o servidor. E se ficou na dúvida qual delas escolher, opte pela opção que tiver menor número. Ex.: Para uma empresa de call center que possua 100 computadores e 200 funcionários trabalhando em turnos diferentes deve optar pela CAL de dispositivo que apresenta menor número
