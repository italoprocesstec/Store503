Nove controles programáveis

 

 

Visão geral

Sob seu comando

Obtenha excelentes resultados com as configurações padrão incluídas no produto. Ou configure gatilhos de um único botão para ações que normalmente requerem vasculhar menus. Ponha comunicações por bate-papo a um fácil alcance. Reduza dpi temporariamente. Reatribua qualquer comando de jogo ou macro multicomando a qualquer um dos nove botões programáveis com o logitech gaming software (lgs) opcional.*

 

* algumas configurações de perfil requerem o logitech gaming software, disponível em gaming.logitech.com/software

 

 

Formato ambidestro esculpido

Conforto é rei

Design versátil em uma forma compacta, este mouse foi construído para conforto e durabilidade

 

 

Iluminação configurável pelo usuário

Conhecimento é poder

Atribua uma cor e uma identidade exclusivas a cada perfil de usuário com até sete variações de de cores para saber quais configurações estão ativas durante um jogo.

 

 

Perfis na memória de bordo

O mouse com cérebro

Configure o mouse uma vez e faça-o memorizar. Cada jogo requer uma configuração exclusiva, seja world of warcraft ou guild wars 2. Programe todos os nove botões para poder levar as configurações para outros computadores.

 

 

Alternância de dpi imediata

Esteja pronto num instante

Faça os movimentos certos em qualquer situação de jogo. Alterne múltiplas configurações de dpi para rapidamente selecionar opções na tela ou examinar o mapa do jogo (até 2500 dpi), ou ainda pormenorizar miras de alvos com precisão de pixel ou selecionar unidades (250 dpi) usando botões localizados de forma conveniente

 

 

Taxa de transmissão de um milissegundo

A ciência da velocidade

Tenha a certeza de que cada comando será executado com a maior rapidez possível. Até oito vezes mais rápido do que um mouse usb padrão, o g300 captura suas jogadas com a mesma rapidez com que você as faz.

 

 

Construção durável

Construído para sobreviver

Os testes de engenharia classificaram os botões principais a dez milhões de cliques e os pés de percurso a 250 km. Capaz de sobreviver às sessões de jogo mais intensas, o g300 pode ir tão longe quanto você

 

 

Pés elegantes

Uma vantagem material

Os pés de politetrafluoroetileno de baixa fricção eliminam a fricção em quase todas as superfícies para proporcionar movimentos mais suaves, movimentos precisos do cursor e exatidão aprimorada

 

 

Software de configuração fácil de usar

Torne-o pessoal - facilmente

Use o logitech gaming software opcional para corresponder comandos perfeitamente aos jogos preferidos. Configurações de arrastar-e-soltar simples permitem que você personalize perfis de botão e rastreamento para qualquer jogo. Ou use personalizações pré-configuradas com detecção de jogo automática
