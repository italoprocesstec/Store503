Criar um sistema de telepresença flexível poderosa

 

 

Visão Geral

 

A Cisco TelePresence SX20 Quick Set é uma solução flexível que ajuda você facilmente transformar qualquer monitor de tela plana em um sistema de telepresença poderosa para pequenas salas de reuniões de tamanho médio. A SX20 Quick Set oferece uma qualidade de alta definição de vídeo, conferência multipartidária, e três opções de câmera diferentes para acomodar o tamanho e as configurações do quarto - tudo a um preço de valor

 

 

Características e Capacidades

 

A SX20 Quick Set fornece o desempenho esperado de sistemas mais caros em um pacote compacto e elegante, multi-featured. Este poderoso conjunto rápido é ideal para pequenas empresas apenas começando com telepresença, ou grandes empresas procurando ampliar os benefícios da telepresença em suas organizações.

 

A SX20 Quick Set pode transformar um monitor de tela plana em um sistema de telepresença em alta definição. Ele também oferece:

 - Incorporado a tecnologia Cisco MultiSite para adicionar mais três participantes a uma chamada

 - Suporte para Cisco Proximidade Inteligente para compartilhamento de conteúdo para dispositivos móveis e compartilhamento sem fio de Macs e PCs *

 - Suporte para conferência comutada para layouts avançados e vídeo em ambas as telas de terminais dual-screen *

 - Simples, conexões intuitivas para configuração simples

 

Além disso, você pode facilmente controlar periféricos quartos como luzes, divisórias, cortinas, e muito mais, directamente a partir do Cisco TelePresence Toque 10 através de uma API personalizável ** (requer um processador de sistema de controle de terceiros externo).
