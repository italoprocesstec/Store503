Gigabit de alto desempenho com rede de 10GbE para pequenas empresas avançadas.

As duas conexões de 10GBASE-T fornecem 10GbE através de cabos de cobre de Categoria 6q para conexões simples, acessíveis, de alta largura de banda, a até 100 metros de distância. Agregue conexões 10GbE para obter largura de banda ainda maior, usando o Protocolo de Controle de Agregação de Links (LACP).

 

 

 

Administração de rede simplificada

Agrupe ou "empilhe" até quatro unidades do switch HP série 1950 de forma lógica e gerencie-os como uma única entidade, simplificando significativamente a administração de rede. Faça a interconexão de unidades empilhadas, usando conexões de 10GbE, com até 40GbE de largura de banda total para conectar as unidades.

 

 

 

Segurança robusta para proteger a rede!

A série de switches HP 1950 permite que usar LANs virtuais e listas de controle de acesso para definir limites quanto a quais dados podem ser vistos por quais usuários. Aproveite o 802.1X com um servidor RADIUS para garantir que apenas usuários conhecidos e autorizados tenham acesso à rede.

 

 

 

Recursos de resiliência para manter a rede funcionando.

Distribua links importantes entre várias unidades em uma pilha lógica, para maior resiliência. As unidades podem até mesmo ficar em prédios diferentes e ainda ser empilhadas logicamente. Dispositivos suportam duas imagens em flash independentes dos sistemas operacionais primário e secundário para backup e recuperação durante atualizações da unidade.
