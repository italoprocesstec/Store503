O que este produto faz

 

O Roteador TL-R470T+ possui alta capacidade de transmissão de dados com alta estabilidade, oferecendo ótimo custo-benefício para locais como "internet cafés" e pequenos escritórios

 

 

Três Portas LAN/WAN Alternáveis

 

O TL-R470T + possui três portas alternáveis que podem ser definidas como LAN ou WAN, permitindo que o roteador suporte até quatro portas WAN para satisfazer as várias solicitações de acesso à Internet através de um único dispositivo. O roteador também tem uma função integrada de balanceamento inteligente de carga, que automaticamente seleciona a linha mais eficiente, de acordo com as solicitações de carga e largura de banda, proporcionando aos usuários um consistente acesso à rede e conectividade confiável via Ethernet. Pelo padrão de fábrica, o TL-R470T + está configurado para funcionar no modo de duas portas WAN: a porta 1 e a porta 2 estão funcionando como portas WAN, enquanto a porta 3 e a 5 são portas LAN.

 

 

Abundantes Recursos de Segurança

 

Para uma defesa contra ameaças externas, TL-R470T+ pode automaticamente detectar e bloquear os ataques de Negação de serviço (DoS - Denial of Service), tais como TCP / UDP / ICMP Flooding, Ping da Morte e outras ameaças relacionadas. Além disso, este roteador oferece funções de filtragem de IP / MAC / URL / Web, o que reforça a prevenção contra-ataques invasores e vírus. Para uma melhor gestão da rede interna, TL-R480T+ permite aos administradores definirem regras para bloquear sites específicos e aplicações IM/P2P com apenas um clique, e restringe o pessoal de usar os serviços específicos, tais como FTP, HTTP e SMTP.

 

 

Gerenciamento Eficiente de Acesso

 

Ao definir um servidor PPPoE para alocar contas PPPoE para os usuários da rede privada, o roteador só permite que os usuários com contas legais, que passaram pela autenticação, possam acessar a Internet. Cooperando com o controle de banda baseado no IP, ele é capaz de confinar o uso da banda de certas contas PPPoE. Além disso, a função de E-Bulletin é capaz de lançar um boletim na forma de página da internet para determinados grupos de usuários e periodicamente notifica os usuários sobre a data de vencimento da conta PPPoE. Com essas características acima, o roteador irá priorizar diferentes usuários em jurisdições de navegação para uma gestão mais eficiente

 

 

Investimentos de Segurança Empresarial

 

Tecnologia de proteção profissional contra raios foi concebida para evitar picos de energia a partir da penetração do interior do dispositivo, para ser descarregado inofensivamente dentro da terra. Este roteador foi projetado para evitar danos causados ​​por relâmpagos de até 4 Kv nas condições de instalação bem fundamentada. Esta característica garante que os investimentos de infra-estrutura de rede permaneçam tão seguro quanto possível dos elementos mais tempestuosos da Mãe Natureza.
