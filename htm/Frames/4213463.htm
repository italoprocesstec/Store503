Projetado para diversas funções, conheça o Srw2048 da Cisco!

O switch Linksys WebView Managed permite-lhe expandir a sua rede de forma segura. A configuração do equipamento é protegida usando SSL para acesso à Web e SSH para o acesso Telnet. Controle de usuário é protegido usando segurança 802.1x através de um mecanismo de autenticação RADIUS e também pode ser controlado usando a filtragem MAC.

 

 

 

Protege o tráfego da rede, mantendo os usuários não autorizados fora da mesma

A Cisco SRW2048 ajuda a proteger a rede usando autenticação de porta 802.1X, listas de controle de acesso (ACLs) e endereços MAC de bloqueio da porta. O padrão 802.1X requer que os clientes se identifiquem antes de passar informações pela rede.

 

 

 

Vantagens WebView

Aproveitando os recursos avançados da Cisco SRW2048 pode proteger ainda mais a integridade da sua rede. A interface WebView oferece uma interface intuitiva interface de gestão, altamente segura, que lhe permite utilizar melhor o conjunto abrangente de recursos do switch para uma rede mais segura e melhor otimizada.
