Ideal para qualquer administrador de rede para criar ou expandir a capacidade de uma rede sem fio

 

 

Visão Geral

 

O DAP-2360 é um 802.11n Plenum PoE Access Point ideal para qualquer administrador de rede para criar ou expandir a capacidade de uma rede sem fio. Este ponto de acesso é usado principalmente em áreas interiores de alto tráfego, como aeroportos, cafés, centros comerciais, espaços desportivos e implantações campus universitário. Com taxas de transferência de até 300 Mbps na faixa de frequência de 2,4 GHz, os usuários são capazes de se conectar com legados adaptadores 802.11b / g, bem como os mais recentes adaptadores 802.11n para desfrutar de downloads mais rápidos e comunicação instantânea
