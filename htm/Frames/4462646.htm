Características

CPU

1. Suporte para Intel ® Core ™ i7 / Intel ® Core i5 ™ / Intel ® Core i3 ™ / Intel ® Pentium ® processadores / Intel ® Celeron ®processadores no pacote LGA1150

2. Cache L3 varia com o CPU

Chipset

1. Intel ® Chipset B85 Expresso

Memória

1. 4 x sockets DIMM DDR3 1.5V com suporte para até 32 GB de memória do sistema 

* Devido a uma limitação do sistema operacional Windows de 32 bits, quando mais de 4 GB de memória física instalada, o tamanho da memória real exibido será menor do que o tamanho da a memória física instalada.

 

2. Arquitetura de memória dual channel

3. Suporte para DDR3 1600/1333 MHz módulos de memória

4. Suporte para módulos de memória não-ECC

5. Suporte para Extreme Memory Profile (XMP) módulos de memória

Gráficos Onboard

Integrated Graphics Processor:

 

1. 1 porta D-Sub x, suportando uma resolução máxima de 1920x1200 @ 60 Hz

2. 1 x porta DVI-D, suportando uma resolução máxima de 1920x1200 @ 60 Hz 

* A porta DVI-D não suporta conexão D-Sub por adaptador.

3. 1 x porta HDMI, suportando uma resolução máxima de 4096 x 2160 @ 24 Hz / 2560 x 1600 @ 60 Hz 

* Suporte para HDMI 1.4a versão.

4. Máximo de memória compartilhada de 1 GB

Áudio

1. Realtek ® ALC892 codec

2. Áudio de Alta Definição

3. 2/4/5.1/7.1 canais 

* Para configurar áudio de 7.1 canais, você tem que usar um módulo de áudio de painel frontal HD e habilitar o recurso de áudio multi-canal pelo driver de áudio.

4. Suporte para S / PDIF

LAN

1. Realtek ® chip de GbE LAN (10/100/1000 Mbit)

Slots de Expansão

1. Slot 1 x PCI Express x16, rodando em x16 (PCIEX16) 

* Para um melhor desempenho, se apenas uma placa gráfica PCI Express é para ser instalado, certifique-se de instalá-lo no slot PCIEX16. 

(O slot PCIEX16 está em conformidade com PCI Express 3.0 padrão .)

 

2. 1 x slot PCI Express x16, rodando a x4 (PCIEX4)

3. Ranhuras 2 x PCI

Tecnologia Multi-Graphics

1. O suporte para a tecnologia AMD CrossFire ™

Interface de Armazenamento

Observação

1. Devido a diferentes condições de suporte para Linux proporcionadas pelos vendedores de chipsets, faça o download do driver Linux pelo website do vendedor de chipset ou site 3o partido.

2. A maioria dos vendedores de hardware / software já não podem oferecer aos motoristas para apoiar Win9X/ME/2000/XP. Se os drivers estão disponíveis a partir dos fornecedores, vamos atualizá-los no site da GIGABYTE.
