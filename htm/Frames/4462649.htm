Especificações

 

CPU

Memória

1. 2 x soquetes de memória DDR3 DIMM que suportam até 16 GB de memória do sistema 

* Devido a uma limitação do sistema operacional Windows de 32 bits, quando mais de 4 GB de memória física instalada, o tamanho da memória real exibido será menor do que o tamanho do físico memória instalada.

 

2. Arquitetura de memória dual channel

3. Suporte para DDR3 1600/1333 MHz módulos de memória

4. Suporte para módulos de memória não-ECC

5. Suporte para Extreme Memory Profile (XMP) módulos de memória

 

(Por favor, consulte a "Lista de memórias suportadas" para mais informações.)
