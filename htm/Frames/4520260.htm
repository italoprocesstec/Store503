Especificações

 

CPU

"1. Suporte para Intel ® Core ™ i7 / Intel ® Core i5 ™ / Intel ® Core i3 ™ / Intel ® Pentium ® processadores / Intel ® Celeron ®processadores no pacote LGA1155

2. Cache L3 varia com o CPU"

 

(Algumas Intel ® Core ™ processadores requerem uma placa gráfica, consulte "Lista de suporte CPU" para mais informações.)

Chipset

1. Intel ® Chipset B75 Expresso

Memória

1. 4 x sockets DIMM DDR3 1.5V com suporte para até 32 GB de memória do sistema 

* Devido à limitação do sistema operacional Windows de 32 bits, quando mais de 4 GB de memória física instalada, o tamanho da memória real exibido será inferior a 4 GB.

 

2. Arquitetura de memória dual channel

3. Suporte para DDR3 2200 (OC) / 1600/1333/1066 MHz módulos de memória 

* Para suportar DDR3 1600 MHz, você deve instalar uma CPU de 22nm da Intel.

4. Suporte para módulos de memória não-ECC

5. Módulos de memória Suporte para Extreme Memory Profile (XMP) 

* Para suportar a memória XMP, você deve instalar uma CPU de 22nm da Intel.

 

(Por favor, consulte a "Lista de memórias suportadas" para mais informações.)

Gráficos Onboard

Integrated Graphics Processor:

1. 1 x porta D-Sub

2. 1 x porta DVI-D, suportando uma resolução máxima de 1920x1200 

* A porta DVI-D não suporta conexão D-Sub por adaptador.

3. 1 x porta HDMI, suportando uma resolução máxima de 1920x1200

Áudio

1. Codec Realtek ALC887

2. Áudio de Alta Definição

3. 2/4/5.1/7.1 canais 

* Para configurar áudio de 7.1 canais, você tem que usar um módulo de áudio de painel frontal HD e habilitar o recurso de áudio multi-canal pelo driver de áudio.

4. Suporte para S / PDIF

LAN

1. 1 x chip Realtek GbE LAN (10/100/1000 Mbit)

Slots de Expansão

1. Slot 1 x PCI Express x16, rodando em x16 (PCIEX16) 

(O slot PCIEX16 conforme a norma PCI Express 3.0.) 

* Para um melhor desempenho, se apenas uma placa gráfica PCI Express é para ser instalado, certifique-se de instalá-lo no PCIEX16 slot. 

* O slot PCI Express x16 suporta até PCI Express 2.0 padrão quando um de 32nm da Intel (Sandy Bridge) CPU está instalado.

2. 1 x slot PCI Express x16, rodando a x4 (PCIEX4) 

(O slot PCIEX4 está em conformidade com PCI Express 2.0 padrão.)

3. Ranhuras 2 x PCI

Tecnologia Multi-Gráficos

1. O suporte para a tecnologia AMD CrossFireX ™
