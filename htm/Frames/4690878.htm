Microsoft Windows Server Standard 2012 R2 x64 P73-06223

Maior flexibilidade e agilidade com o Windows Server 2012 R2

 

 

Visão Geral

 

No centro da visão do Microsoft Cloud OS, o Windows Server 2012 R2 traz a experiência da Microsoft entregando serviços na nuvem em escala global à sua infraestrutura com novos recursos e melhorias em virtualização, gerenciamento, armazenamento, rede, infraestrutura de área de trabalho virtual, acesso e proteção das informações, a Web e a plataforma de aplicativos e muito mais.

 

Virtualização de servidor

 

Tire proveito das economias de custo da virtualização e maximize os investimentos em hardware de servidor consolidando seus servidores como máquinas virtuais em um único host físico. O Hyper-V executa vários sistemas operacionais, incluindo o Windows, o Linux e outros, em paralelo, em um único servidor. O Windows Server 2012 R2 estende os recursos do Hyper-V com recursos adicionais e escalabilidade líder do setor para processadores e memória do host

 

Armazenamento

 

Seja qual for sua plataforma de armazenamento, os dados que ela mantém são a essência de seus negócios. O Windows Server 2012 R2 ajuda você a otimizar seus investimentos em armazenamento existentes, como SANs. Também permite que você crie soluções de armazenamento de alto desempenho e de alta disponibilidade usando hardware padrão do setor e o Windows Server 2012 R2. O Windows Server 2012 R2 ajuda a garantir que seu armazenamento permaneça continuamente disponível, de forma que seus serviços possam estar continuamente disponíveis também

 

Rede

 

Você pode gerenciar uma rede inteira como um único servidor, para obter confiabilidade e escalabilidade de vários servidores a um custo mais baixo. O redirecionamento automático em torno de falhas no armazenamento, no servidor e na rede mantém os serviços de arquivos online com visibilidade mínima do tempo de inatividade. Junto com o System Center 2012 R2, o Windows Server 2012 R2 pode fornecer uma solução completa de rede definida por software entre implementações de cloud pública, privada e híbrida

 

Gerenciamento e automação do servidor

Seguindo a abordagem de gerenciamento baseado em padrões, o Windows Management Framework fornece uma plataforma comum de automação e integração para ajudar você a automatizar suas tarefas de rotina com ferramentas como o Windows PowerShell. Outros aprimoramentos ajudam a simplificar a implantação, garantir que os componentes de seus datacenters tenham a configuração correta e permitem que você tome a iniciativa de gerenciar vários servidores por meio de um único painel relevante no Gerenciador de Servidores.

 

Plataforma Web e de aplicativos

 

O Windows Server 2012 R2 se baseia na tradição da família Windows Server como uma plataforma de aplicativos comprovada, com milhares de aplicativos já criados e implantados e uma comunidade já estabelecida de milhões de desenvolvedores reconhecidos e habilitados. Você pode criar e implantar aplicativos no local ou na nuvem, ou nos dois lugares, com soluções híbridas que funcionam nos dois ambientes

 

Proteção das informaçõe

 

Com as soluções de Proteção de Informações da Microsoft, você pode gerenciar uma única identidade para cada usuário entre aplicativos locais e baseados na nuvem (SaaS). Você define o nível de acesso de cada usuário às informações e aplicativos com base em quem os usuários são, o que estão acessando e qual dispositivo estão usando, até aplicando autenticação multifator. Você pode fornecer acesso remoto seguro a seus usuários móveis usando os recursos de Acesso remoto (RRAS) do Windows Server com recursos como DirectAccess e VPN (incluindo conexões VPN automáticas) e permitir que seus usuários sincronizem seus arquivos de trabalho em um servidor corporativo com seus dispositivos. Você também pode gerenciar dispositivos móveis para remover dados e aplicativos corporativos quando um dispositivo for perdido, roubado ou desativado

 

Virtual Desktop Infrastructure (VDI)

 

Com o Windows Server 2012 R2, é ainda mais fácil implantar e entregar recursos virtuais entre dispositivos. As tecnologias de VDI oferecem acesso fácil a um ambiente sofisticado e de alta fidelidade do Windows no datacenter, de virtualmente qualquer dispositivo. Por meio do Hyper-V e dos Serviços de Área de Trabalho Remota, a Microsoft oferece três opções flexíveis para a implantação de VDI em uma única solução: Áreas de Trabalho em Pool, Áreas de Trabalho Pessoais e Sessões de Área de Trabalho Remota
