Posicionamento Flexível Design para Fácil Montagem e PoE

 

 

Visão Geral

 

A aparência de “lâmpada de teto” do EAP bem como seu design para fácil instalação faz deste produto perfeito para se integrar com qualquer parede ou teto a fim de se misturar com a maioria das decorações. A série de EAPs suporta o Sistema PoE para torná-los ainda mais flexíveis com seus posicionamentos

 

 

Nível Corporativo Wireless

 

Hardware de Alta Qualidade

O EAP possui tecnologia Wi-Fi 802.11n e MIMO 2*2 – o EAP220 suporta duas bandas simultaneamente de 2.4GHz e 5GHz com velocidades de 600Mbps enquanto o EAP120 desempenha velocidades de até 300Mbps

A porta Ethernet Gigabit garante acesso à Internet

De alta velocidade

Com chipsets Qualcomm Atheros de 560MHz,

O EAP garante incrível cobertura wireless e confiabilidade

 

 

Gerenciamento Wi-Fi Centralizado

 

Software para Gerenciamento

 

O software gratuito do controlador permite ao admistriador que ele gerencie até centenas de EAPs através de um único IP como um Controlador de Acesso.

*For EAP120 that’s currently supporting cluster function, please upgrade the device’s firmware to the latest to be compatible with the EAP controller software

 

 

Portal Cativo para Autenticação Eficiente de Convidado

O Portal Cativo oferece um método conveniente de autenticação para convidados Wi-Fi. A ação solicitada pode ser verificada por um servidor de portal externo ou uma base de dados personalizad apelo administrador, que pode ser simplesmente visualizar e concordar com as políticas de uso ou até mesmo inserir o ID e a senha

 

 

Aplicável ao Ambiente com Múltiplos Usuários

Desenhado para Desempenho Corporativo

 

Com seu design para fácil montagem, fornecimento de energia PoE, funão de agrupamento, software para gerencimento centralizado, Portal Cativo, Autogerenciamento RF e características físicas de nível corporativo, a série de EAPs foram feitas para ambientes exigentes como schools, hotéis e empresas
