Informação do Produto

 

Series Switches 2960-X Cisco® Catalyst® são de configuração fixa, switches Gigabit Ethernet empilháveis ​​que fornecem acesso de classe empresarial para aplicações de campus e filiais

Especificação Técnica

 

- Tipo de dispositivo:

Interruptor - 24 portas - gerenciado.

- Tipo de caixa:

Montagem em rack.

- Portos:

24 x 10/100/100.

- Desempenho: Capacidade de comutação:

100 Gbps - desempenho de encaminhamento: 71,4 Mpps.

- Características:

Gigabit Small Form-Factor Pluggable (SFP), FlexStack Plus para empilhamento de até 8 interruptores com 80 Gbps de taxa de transferência de pilha (opcional), Cisco EnergyWise, Cisco Auto QoS, Quality of Service (QoS), Cisco Smart Install, Cisco Auto SmartPorts , Cisco Smart Troubleshooting, Power over Ethernet PoE (IEEE 802.3af): 24 portas até 15,4 W, PoE + (IEEE 802.3af): 12 portas de até 30W - Disponível PoE: 370W, Dynamic Host Configuration Protocol (DHCP), Autonegotiation, Dinâmica Trunking Protocol (DTP), Port Aggregation Protocol (PAgP), Link Aggregation Control Protocol (LACP), interface de passagem de dependentes de mídia automático (MDIX), unidirecional link Detection Protocol (UDLD), comutação Database Manager (SDM), local Endereço Proxy Resolution Protocol (ARP), o Smart Multicast, com tecnologia Cisco FlexStack-Plus, Internet Group Management Protocol (IGMP), Multicast VLAN Registration (MVR), Per-port broadcast, multicast e unicast controle tempestade, VLAN de voz, Cisco VLAN Trunking Protocol (VTP), Switch Porta remota Analyzer (RSPAN), Layer 2 rota do traço, Trivial File Transfer Protocol (TFTP), Timing Network Protocol (NTP).
