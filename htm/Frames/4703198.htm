O HD externo para desktop, com alto armazenamento extra e facilidade de uso. Confira!

O disco Seagate® Expansion para desktop oferece armazenamento adicional para a sua crescente coleção de arquivos. Adicione espaço para mais arquivos instantaneamente, reúna todos os seus arquivos em um só local ou libere espaço do disco interno do seu computador para aumentar o desempenho.

A instalação é simples, bastando conectar o cabo de fonte de alimentação e o cabo USB incluído e pronto. Ele é reconhecido automaticamente pelo sistema operacional Windows®, sem necessidade de instalar software e sem nada a configurar. Salvar arquivos também é fácil — é só arrastar e soltar.

Aproveite ao máximo as velocidades de transferência de dados rápidas da interface USB 3.0 conectando o disco a uma porta USB 3.0 SuperSpeed. A interface USB 3.0 é compatível retroativamente com USB 2.0, proporcionando maior flexibilidade.
