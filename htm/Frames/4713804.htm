Características

 

Vá além da virtualização 

Windows Server 2012 R2 é uma plataforma de virtualização completa, que proporciona um ambiente totalmente isolado, multi-tenant. Ele inclui ferramentas para ajudar a contratos de garantia de nível de serviço (SLAs), permite que os estornos através de faturamento com base em uso, e suporta a entrega de auto-atendimento.

Aproveite o poder de muitos servidores com a simplicidade de uma única plataforma

armazenamento flexível. Melhorar o desempenho, disponibilidade e eficiência dos recursos com o armazenamento de sua escolha.

Benefício de uma plataforma com aplicativos já criados e implantados

Windows Server 2012 R2 permite que você construa aplicações simétricas híbrido e em todo o centro de dados e da nuvem.

Flexível de acesso do usuário a dados e aplicações

Acesso a partir de praticamente qualquer lugar, a partir de qualquer dispositivo. , Acesso Seamless sob demanda para um ambiente de trabalho virtualizado a partir de praticamente qualquer lugar, incluindo filiais e serviços de conectividade públicas.
