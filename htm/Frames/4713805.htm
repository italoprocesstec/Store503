Faça mais com menos? A parceria HP e Microsoft estende Windows Server 2012 apresenta com o lançamento do Windows Server 2012 R2.
 

Combinando as características HP ProLiant Gen8 com as virtualização e escalabilidade recursos avançados do Windows Server 2012 R2 permite que usuários de TI e fornece acesso flexível a recursos corporativos ao mesmo tempo proteger informações corporativas. Editions: - Windows Server 2012 R2 Datacenter: para ambientes de nuvem privada altamente virtualizados - Windows Server 2012 R2 Standard: para ambientes físicos ou minimamente virtualizados - Windows Server 2012 R2 Essentials: uma oferta única cloud-enabled para até 25 usuários - Windows Server 2012 R2 Foundation: experiência de Windows Server para até 15 usuários Este novo aumenta a liberação virtualização de servidores, armazenamento, gerenciamento de rede e suporte de plataforma de aplicação.
 

Vá além da virtualização

 

Windows Server 2012 R2 é uma plataforma de virtualização completa que oferece um ambiente totalmente isolado, multi-tenant. Ele inclui ferramentas para ajudar os acordos de garantia de nível de serviço (SLAs), permite que os estornos através de faturamento com base em uso, e suporta a entrega de auto-atendimento.

Aumentar a escalabilidade e desempenho por meio de uma alta densidade, ambiente escalável que pode executar de forma otimizada, uma vez que se adapta às suas necessidades de mudança.

Obter conectividade para serviços de nuvem usando uma identidade comum e uma estrutura de gestão para, a conectividade entre locais com segurança avançada confiável.
 

Aproveite o poder de muitos servidores com a simplicidade de uma única plataforma

Armazenamento flexível. Melhorar o desempenho, a disponibilidade e a eficiência dos recursos com o armazenamento de sua escolha.

Disponibilidade contínua. Fornecer, serviços altamente disponíveis de custo-benefício com proteção contra uma ampla gama de cenários de falha - para um único servidor ou todo o data center.

Eficiência da gestão. Automatizar um amplo conjunto de tarefas de gestão e simplificar a implantação de funções de servidor como você se move em direção, automação lights-out completo.
 

Benefício de uma plataforma com aplicativos já construído e implantado

Windows Server 2012 R2 permite que você construa aplicações simétricas híbrido e em todo o data center e a nuvem.

Fornece estruturas, serviços e ferramentas para aumentar a escalabilidade e elasticidade para aplicações e sites que são multi-tenant consciente.

Construa aplicativos de missão crítica e tirar proveito dos quadros abertos, aplicativos de código aberto, e várias linguagens de desenvolvimento.
 

Flexível de acesso do usuário a dados e aplicações

Acesso a partir de praticamente qualquer lugar, a partir de qualquer dispositivo. Acesso contínuo on-demand para um ambiente de trabalho virtualizado a partir de praticamente qualquer lugar, incluindo filiais e serviços de conectividade públicas.

Experiência completa do Windows em qualquer lugar. A rica experiência de usuário personalizada sobre dispositivos populares, enquanto se adapta a diferentes condições de rede de forma rápida e responsavelmente.

Reforçada a segurança de dados e conformidade. Políticas de auditoria e de acesso centrais permitir o acesso granular aos dados e recursos corporativos baseados em forte identidade, a classificação de dados e administração simplificada para acesso remoto.
