Sistema operacional

Windows 7 Professional 64 (disponível através de direitos de downgrade do Windows 8.1 Pro)

(Este sistema vem com o Windows 7 Professional instalado e também com uma licença e uma mídia do Windows 8.1 Pro. Você pode usar apenas uma versão do Windows por vez. Para alternar entre as versões, você terá que desinstalar uma versão e instalar a outra. Você deve fazer o backup de todos os seus dados (arquivos, fotos, etc.), antes de desinstalar e instalar sistemas operacionais, para evitar perda de dados.)
