Informações do produto

 

Os varejistas  perceberam que agora é o momento de impulsionar a transformação,das suas lojas, tornando suas operações mais inteligentes e competitivas. Aqueles que investem em tecnologias que reduzem os custos, oferecendo eficiências operacionais estarão prontos para o futuro. Para estes varejistas, a impressora de uma estação Toshiba 1NR é a melhor opção. Criada para proporcionar um desempenho ótimo e robusto em ambientes de varejo exigentes, as impressoras Toshiba oferecem velocidade, confiabilidade e capacidade inigualável de gerenciamento e manutenção. 

As impressoras Toshiba são especificamente projetadas para atender aos rigorosos ambientes de varejo e possuem as tecnologias de impressão mais avançadas da atualidade para agilizar as transações. 

Características

 

1. Impressões rápidas de recibos a 80 LPS ;

2. Abertura da tampa por botão de pressão;

 

3. Fácil carregamento de papel (drop-in) e grandes rolos de recibo com 102 mm (4 pol.);

4. O design robusto e resistente a derramamentos proporciona alta confiabilidade nos ambientes de varejo mais rigorosos; 

5. O suporte à página de códigos estendida (27 no total);

6. Memória flash do usuário de 3 MB, que suporta códigos de barras, logotipos, gráficos coloridos e fontes TrueType™ ;

7. Um alarme sonoro programável pode ser usado para emitir avisos;

8. Sensor de atolamento de papel ,sensores do nível de papel, Sensores do estado da cabeça de impressão;

9. Todos os periféricos da Toshiba possuem coordenação de cores e conexões com travas, além de serem testados, desenvolvidos, compatíveis e otimizados para funcionar como uma solução completa para a loja; 

10. Os conectores RS-485 e USB com alimentação eliminam a necessidade de um dispositivo separado de alimentação e economizam espaço no balcão; 

11. Os conectores RS-232 e USB padrão integram o produto a sistemas de POS ;

12. Kit de homologação fornecido pela Toshiba;

13. Todos os materiais da embalagem são recicláveis; 

14. Um suporte de parede opcional pode ser usado para economizar espaço valioso no balcão; 

15. Disponíveis nas cores branco e iron gray.

- RS-232: Direct I / O

● USB e RS-485: V5R2 e V6R2 ** Microsoft Windows XP, Microsoft Windows Embedded para Point of Service (WEPOS) 1.11, Microsoft Windows 2009 POSReady Incorporado, Microsoft Windows Embeded POSReady 7, Microsoft Windows 7 (Professional / Ultimate):

- RS-232: Direct I / O, o Windows drivers nativos (NWD), OPOS, JavaPOS

- RS-485: OPOS, JavaPOS

- USB: o Windows drivers nativos (NWD), OPOS, JavaPOS

● IBM Retail Environment para SUSE Linux Enterprise Desktop (SLED) 11, SUSE Linux Enterprise Server (SLES) 11, SUSE Linux Enterprise Point of Service (SLEPOS) 11:
