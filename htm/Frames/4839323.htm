Windows Server 2012 Foundation, inovação e praticidade

Windows Server 2012 Foundation viabiliza recursos de TI essenciais, tais como compartilhar arquivos e impressoras, acesso remoto e segurança.

Ele fornece uma base de rede a partir do qual você pode gerenciar centralmente as configurações em seus computadores que são baseados no sistema operacional Windows
