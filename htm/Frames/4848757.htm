Gerenciamento de energia inteligente para os seus equipamentos

O Edx1000Hb possui em sua estrutura o software de monitoramento WinPower Ups, o qual fornece uma interface fácil de usar para monitorar e controlar a sua UPS. Este software exclusivo fornece proteção de energia completa para o seu computador durante uma falha de energia. Com ele, os usuários podem monitorar qualquer estado da UPS na mesma rede local. Além disso, a UPS pode fornecer proteção de segurança para mais de um computador na mesma rede local, ao mesmo tempo, como desligar o sistema de segurança, poupando os dados de aplicativos e desligar o UPS quando o poder falhar.
