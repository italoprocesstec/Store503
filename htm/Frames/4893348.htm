Construído para 24/7 always-on de vigilância na segurança de alta definição sistemas que usam até oito discos rígidos e até 32 câmeras

 

 

Visão Geral

 

Armazenamento Vigilância Roxo WD é construído para 24/7 always-on de vigilância na segurança de alta definição sistemas que usam até oito discos rígidos e até 32 câmeras. Tecnologia exclusiva AllFrame ™ trabalha com streaming de dados para reduzir pixelation erro e vídeo interrupções que ocorrem quando os discos rígidos de desktop são usados incorretamente como armazenamento em sistemas de segurança

 

 

Armazenamento superior. Vigilância superior.

As unidades WD Purple são construídas para sistemas de segurança de alta definição de funcionamento contínuo. Com suporte para até 8 unidades de disco rígido e 32 câmeras, as unidades WD Purple são otimizadas para vigilância. O armazenamento para vigilância WD Purple vem também com a tecnologia AllFrame, exclusiva da WD, para que você possa criar com confiança um sistema de segurança personalizado para as necessidades da sua empresa. Usando a tecnologia AllFrame, as unidades WD Purple aprimoram o streaming ATA para reduzir os erros, a pixelização e as interrupções de vídeo que acontecem quando a unidade errada é usada em seu sistema de gravação de vídeo.

 

 

A unidade certa para o trabalho certo.

As unidades WD Purple foram projetadas especificamente para a vigilância, por isso elas são capazes de resistir a extremas flutuações de calor e a vibrações do equipamento que ocorrem em um ambiente de DVR. Uma unidade de desktop média é projetada para operar apenas em intervalos curtos, não no ambiente agressivo, de operação contínua, de um sistema de vigilância de alta definição. Uma interrupção em uma unidade poderia resultar em uma falha com impacto em vários usuários, em seus arquivos de segurança ou até mesmo em todo o sistema. Com o WD Purple, você tem armazenamento confiável próprio para vigilância testado para ser compatível com uma ampla variedade de sistemas de segurança.

 

 

Armazenamento líder da indústria. Vigilância em que você pode confiar.

A WD é líder mundial na indústria de discos rígidos. Com o armazenamento para vigilância WD Purple, você tem uma unidade projetada para sistemas de vigilância de operação contínua, de alta temperatura, para que você possa confiar na reprodução de vídeo de qualidade quando precisar mais. Quer você esteja protegendo os entese queridos ou monitorando sua empresa, o WD Purple oferece o desempenho em que você pode confiar

 

 


Expanda sua visão para 32

Ampla compatibilidade. Integração simplificada.
