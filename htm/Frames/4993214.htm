Visão Geral

 

Licença Perpétua do Microsoft Windows Server CAL 2012 - Licenças para acesso ao Server - Licenciamento por máquina

 

Se você precisa licenciar um computador para ter acesso ao servidor, adquira uma licença de Windows Server Device Cal 2012 para acessar os sistemas Windows Server DataCenter 2012, Windows Server Standard 2012 ou sistemas de Windows Server anteriores. A regra geral é que todo acesso ao servidor precise de uma Cal.

A licença é associada a um dispositivo cliente, que pode ser utilizado por vários usuários para acessar o produto, serviço ou tecnologia do servidor.

 

Exemplo: Se 50 estações de trabalho acessam um servidor com Windows Server 2012 com 150 usuários trabalhando nestes equipamentos, a melhor solução de licenciamento de CALs é a empresa adquirir 50 CALs de dispositivos do Windows Server 2012.
