Sites e aplicativos

 

A solução Sites do Azure é uma PaaS (Plataforma como Serviço) totalmente gerenciada que permite criar, implantar e dimensionar aplicativos Web de nível empresarial em questão de segundos. Concentre-se no código de seu aplicativo e deixe o Azure cuidar da infraestrutura para dimensioná-lo e executá-lo com segurança.

 

 

Recursos poderosos

 

Sites + Hospedagem na Web - Implante e dimensione sites e aplicativos Web modernos em segundos;

Máquinas virtuais - Abra o Windows Server e o Linux em questão de segundos;

Banco de dados SQL - SQL Relacional Gerenciado Banco de Dados como um Serviço;

Aprendizado de Máquinas - Análise preditiva poderosa baseada em nuvem;

Back-ends móveis - Compilar e hospedar o back-end para qualquer aplicativo móvel;

RemotApp - Implante aplicativos de clientes do Windows na nuvem, execute em qualquer dispositivo.

Suporte a uma escala flexível e um desempenho mais previsível

 

 

Apoiados pela potência e presença do Azure, aplicativos de alta taxa de transferência dimensionar elasticamente sua camada de dados em milhares de bancos de dados. As novas camadas de serviço do Banco de Dados SQL permitem que aplicativos aumentem ou diminuam seu desempenho previsível em cada banco de dados. A nova visualização Escala Elástica para o Banco de

Dados SQL do Azure simplifica o desenvolvimento e o gerenciamento de aplicativos que se expandem em um grande número de bancos de dados
