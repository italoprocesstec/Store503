O sistema que é a ferramenta poderosa e completa, ideal para pequenas e médias empresas. Confira!

Ideal para empresas que querem ter um ambiente de nuvem privada e híbrida altamente virtualizado. Como sempre, ele fornece acesso a todos os recursos do produto e permite ilimitadas instâncias do Windows Server com cada licença, permitindo que seu ambiente virtual para crescer como você faz. O licenciamento para a edição Standard continuará a ser processador mais CAL (Client Access License), com cada licença que cobre até dois processadores físicos em um único servidor.

O Windows Server Standard 2012 R2 tem acesso ilimitado de usuários (CALs necessárias), licencia até 2 processadores por licença, pode virtualizar até 2 máquinas.

No esplendor da visão do sistema operacional em nuvem da Microsoft, O Windows Server 2012 Standard traz a experiência da Microsoft entregando em escala global serviços em nuvem para sua infraestrutura. Windows Server 2012 Standard oferece diversas opções e ferramentas através de virtualização, armazenamento, conexão, infraestrutura de desktop virtual, acesso e proteção de dados, e mais.

O Windows Server DataCenter 2012 R2 fornece uma grande variedade de recursos novos e aprimorados que abrangem virtualização de servidores, armazenamento, rede definida por software, gerenciamento e automação de servidores, plataforma Web e de aplicativos, proteção de acesso e informações, infraestrutura de desktop virtual e muito mais.
