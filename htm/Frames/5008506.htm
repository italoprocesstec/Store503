O Windows Server 2012 R2 Essentials é uma solução de servidor flexível, acessível e fácil de usar desenvolvido e com preço para pequenas empresas.

O servidor ideal para pequenas empresas, o Windows Server 2012 R2 Essentials simplifica a integração de aplicativos e serviços baseados na nuvem opcionais, como o Microsoft Office 365 e Microsoft Azure. O Windows Server 2012 R2 Essentials pode ajudar a minimizar o tempo, o esforço e o dinheiro que você gasta em TI.

 

Proteja seus dados

Os backups do servidor são configurados com um assistente simples. Você também pode tirar proveito de serviços de backup opcionais baseados na nuvem, como o Backup do Microsoft Azure, para obter um nível maior de proteção. Os backups de cliente são configurados para serem executados diariamente para cada PC conectado ao ambiente do Essentials. Os dados pessoais podem ser protegidos facilmente com o Histórico de Arquivos, um novo recurso de clientes Windows 8, que pode ser configurado centralmente e gerenciado em seu servidor do Essentials.

 

Serviços integrados na nuvem

Quando integrado com o Office 365, você pode criar as contas de seus usuários no Windows Server 2012 R2 Essentials e uma caixa de correio associada é criada ao mesmo tempo. As senhas dos usuários são sincronizadas entre o servidor Essentials e o serviço Office 365 de forma que os usuários entram da mesma maneira, seja para acessar recursos locais ou na nuvem.

 

Forneça acesso remoto seguro

 

Existem várias opções de acesso remoto, incluindo RWA (Acesso via Web Remoto), uma maneira segura dos usuários e administradores acessarem arquivos e pastas, computadores cliente e servidores. O aplicativo My Server para Windows 8 e Windows 8 RT é uma maneira amigável de acessar seus arquivos e pastas, além de executar tarefas administrativas leves em desktops, laptops e dispositivos tablet. O aplicativo My Server para Windows Phone 7 e Windows Phone 8 fornece uma opção móvel.

 

Virtualização de servidor

As pequenas organizações desejam se concentrar em seu negócio principal, não em gerenciar uma infraestrutura de TI. O Windows Server 2012 R2 Essentials pode ajudar a minimizar o tempo, o esforço e o dinheiro que você gasta em TI.

 

Implantação do servidor

Entre as opções de implantação do servidor do Windows Server Essentials estão:

O Windows Server Essentials Experience como uma função de servidor nas edições Standard e Datacenter do Windows Server 2012 R2 pode ser instalado como um servidor membro em um domínio.

O Windows Server 2012 R2 Essentials pode ser instalado em um servidor físico ou como uma máquina virtual.

 

Implantação do cliente

O Windows Server 2012 R2 Essentials oferece opções melhoradas de implantação do cliente:

O Connector pode ser usado na conexão com um servidor no qual o Windows Server 2012 R2 Essentials esteja em execução em um local remoto.

O Connector pode disparar a discagem VPN automática em clientes que ingressaram em um domínio, de maneira que os clientes possam estar sempre conectados ao Windows Server 2012 R2 Essentials quando estiverem trabalhando fora do local.

O Connector pode ser configurado para se conectar a outros servidores nos quais o Windows Server 2012 R2 Essentials esteja em execução sem reinstalação.

O desempenho do Connector foi melhorado com a redução da memória e do uso da CPU.

 

Usuários e grupos

As seguintes funcionalidades foram adicionadas para gerenciar usuários e grupos:

Gerenciamento de grupos de usuários: é possível criar grupos e adicionar contas de usuário a eles. As permissões de conta de usuário podem ser gerenciadas por meio das associações a grupos de usuários.

Gerenciamento de grupos de distribuição do Office 365: caso o servidor esteja integrado ao Office 365, é possível gerenciar os grupos de distribuição do Office 365 por meio do Painel.

 

Armazenamento do servidor

As seguintes funcionalidades foram adicionadas ao armazenamento do servidor:

É possível criar pastas compartilhadas em um servidor secundário ao mesmo tempo em que se mantém o mesmo prefixo do campo UNC (Universal Naming Convention, Convenção de nomenclatura universal).

É possível especificar uma cota de espaço para uma pasta do servidor. Você recebe um alerta quando o tamanho de uma pasta do servidor cresce além da cota definida.

 

Proteção de dados

O Windows Server 2012 R2 Essentials inclui um Histórico de Arquivos melhorado.

Os backups do Histórico de Arquivos são configurados por usuário, e não por dispositivo. Além disso, é possível restaurar automaticamente os dados por meio de backups do Histórico de Arquivos em um computador, caso o computador anterior não seja mais funcional ou esteja indisponível. Na primeira vez em que fizer logon no novo computador na rede, você deverá restaurar os arquivos usando os backups do Histórico de Arquivos.

 

Relatórios de Integridade

Os Relatórios de Integridade estão integrados ao Windows Server 2012 R2 Essentials, e não precisam mais ser instalados como um suplemento.

A Página inicial inclui as guias Alertas e Relatórios. Dependendo das necessidades comerciais, é possível personalizar os relatórios de integridade do sistema para exibir itens que você deseja monitorar.

 

BranchCache

O BranchCache está integrado ao Windows Server 2012 R2 Essentials.

Será possível ativar o BranchCache para melhorar o acesso aos dados se o servidor no qual o Windows Server 2012 R2 Essentials esteja em execução estiver fora do local.

 

Integração do Office 365

As seguintes funcionalidades estão disponíveis quando o Office 365 está integrado ao Windows Server 2012 R2 Essentials:

Gerenciamento de bibliotecas do SharePoint

 

Gerenciamento de grupos de distribuição do Office 365

Importante: a integração do Windows Server 2012 R2 Essentials ao Office 365 só tem suporte em um ambiente com um único controlador de domínio. O Assistente de Integração do Office 365 deve ser executado em um controlador de domínio.


Microsoft Windows Server 2012 R2 Essentials Oem G3S-00710

 

Dispositivos móveis

É possível gerenciar dispositivos móveis por meio do painel.

Caso o servidor esteja integrado ao Office 365, é possível gerenciar os dispositivos móveis usando-se as funcionalidades do Exchange Active Sync. Por exemplo, você pode definir o acesso ao email em um dispositivo móvel, configurar políticas de senha e realizar uma limpeza remota do dispositivo móvel.

 

Restauração completa do sistema do cliente

A restauração completa do sistema do cliente é compatível com os Serviços de Implantação do Windows.

É possível criar um serviço de restauração do cliente usando-se a tarefa Configurar Serviço de Restauração do Cliente e realizar uma restauração completa do sistema do cliente pela rede, em vez de usar uma imagem salva em um DVD.

 

Acesso remoto à Web

O Acesso Remoto via Web é otimizado para dispositivos sensíveis ao toque e avançados com suporte a HTML5.

 

Integração do Microsoft Online Services

A integração do Windows Server 2012 R2 Essentials ao Microsoft Online Services, inclusive o Microsoft Azure Active Directory (Azure AD), o Office 365, o Windows Intune e o Exchange no local, só é compatível em um ambiente com um único controlador de domínio. O assistente de integração do Microsoft Online Services deve ser executado em um controlador de domínio.
