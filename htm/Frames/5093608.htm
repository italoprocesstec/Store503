Visão Geral

 

O processador Dell Intel Xeon® E5-2620 v3 de seis núcleos de 2,40 GHz melhora a performance e a velocidade do seu sistema. Além disso, a tecnologia de virtualização permite a migração de mais ambientes. Oferece suporte à tecnologia SpeedStep® avançada, que permite compensações entre a performance e o consumo de energia.

 

Suas tecnologias de monitoramento térmico protegem o pacote do processador e o sistema contra falhas que acontecem pelo superaquecimento de diversos recursos de gerenciamento térmico. O recurso VT-x com Extended Page Tables (EPT), também conhecido como Conversão de Endereços de Segundo Nível (SLAT), fornece aceleração para aplicativos virtualizados com uso intenso de memória. Com mais recursos, o processador Xeon E5-2620 v3 de 2,40 GHz é a opção ideal para praticamente todos os seus aplicativos de infraestrutura empresarial padrão ou com uso intenso de dados

 

 

Destaques

 

Com processador de 2,40 GHz para alta transferência de dados

Velocidade o barramento de 8 GT/s para processamento rápido

Oferece um cache de 15 MB para acesso mais rápido aos dados

Baseando em tecnologia de processador de 22 nm para performance avançada

O recurso Execute Disable Bit permite que o processador marque a memória como executável ou não executável
