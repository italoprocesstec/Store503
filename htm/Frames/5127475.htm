Fornecer energia limpa para dispositivos com suporte PoE + rede

A Dell Networking N2000 Series tem opções para até 48 portas de Power-over-Ethernet Plus (PoE +). Isso permite que você conectar dispositivos com requisitos de alta potência (até 30,8 watts) sem executar um cabo de alimentação separado. PoE + é particularmente útil em edifícios mais antigos, onde a instalação de energia para vários locais, difundidas pode ser proibitivamente caro.

 

Projetado para a eficiência

 

Criada para ajudar a manter os custos operacionais para baixo, a série N2000 pode operar em temperaturas de até 113 ° F (45 ° C) para ajudar a reduzir os custos de resfriamento. Recursos projetados para melhorar a eficiência geral incluem:

 

 80PLUS certificado eficiente fonte de alimentação integrada

 Ventiladores de velocidade variável redundantes

 Ethernet e de baixo consumo de energia PHYs-Efficient para ajudar a reduzir a potência para inativo portos e das ligações ociosas
