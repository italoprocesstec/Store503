Quarta geração da linha de produtos processadores core i3

 

 

Visão Geral

 

O i3-4170 3.7 GHz de processador Dual-Core Núcleo de Intel é a quarta geração da linha de produtos processadores core i3, que agora é construído usando a arquitetura Haswell. Este processador tem uma frequência base de 3,7 GHz e como acontece com muitas tecnologias Haswell, vem com recursos como Hyper-Threading. Este processador também possui dois núcleos com quatro threads em uma tomada FCLGA1150 e tem 3 MB de memória cache. Tendo dois núcleos permite que o processador para executar vários programas simultaneamente sem abrandar o sistema, enquanto os quatro tópicos permitir que os aplicativos para executar as tarefas mais cedo, trabalhando em paralelo. Este Intel Core i3-4170 3.7 GHz Dual-Core Processor é bem adequado para usuários que está interessado em tarefas de uso geral, tais como navegação na Internet, checar e-mails, criação de documentos e edição, reprodução de multimídia e muito mais. Este processador também suporta até 32 GB de memória DDR3 e DDR3L 1333/1600 MHz RAM, tem integrado Intel HD Graphics 4400, e tem SSE 4.1 / 4.2 e 2.0 AVX extensões, bem como as novas instruções AES
