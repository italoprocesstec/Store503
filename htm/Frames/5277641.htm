Visão geral

As unidades WD roxas são construídos para 24/7, always-on, sistemas de segurança de alta definição. Com suporte para até 8 discos rígidos e 32 câmeras, as unidades WD roxas são otimizados para a vigilância. Armazenamento de vigilância WD roxo também vem com a tecnologia allframe exclusiva da WD, de modo que você pode confiantemente criar um sistema de segurança sob medida para as necessidades do seu negócio. Usando allframe tecnologia, as unidades WD roxas melhorar ATA streaming para reduzir os erros, pixilation, e interrupções de vídeo que acontecem quando a unidade errada é usado em seu sistema de gravador de vídeo

 

 

A unidade certa para o trabalho certo

WD unidades roxas foram projetadas especificamente para a vigilância, por isso eles são capazes de suportar a extrema flutuações de calor e vibrações do equipamento dentro de um ambiente DVR. Uma unidade de área de trabalho média é construída para ser executado por apenas curtos intervalos de tempo, não a dura 24/7 always-on ambiente de um sistema de vigilância de alta definição. A interrupção de uma unidade pode resultar em uma falha que afeta vários usuários, o seu arquivo de segurança, ou até mesmo todo o sistema. Com WD roxo, você começa, armazenamento de vigilância de classe confiável, que é testado para compatibilidade em uma ampla gama de sistemas de segurança
