Gerenciador de console de próxima geração que fornece acesso remoto seguro para dispositivos de TI e servidores UNIX / baseados em Linux

 

 

Visão Geral

 

A série servidor consola ACS 6000 integra tecnologias avançadas, serviços de adaptação e de comunicações empresariais seguras para high-end, grandes centros de dados empresariais. Estes servidores de console oferecer o desempenho ea confiabilidade necessária para acessar de forma eficiente e gerenciar ativos de TI, como servidores (UNIX, Linux e Solaris), roteadores, switches, equipamentos de troca ramo público (PBX) de telecomunicações (DSU / CSU) e dispositivos de energia de série.

 

A arquitetura da série servidor consola ACS 6000 de alto desempenho oferece recursos avançados de hardware e software para atender os exigentes requisitos de aplicações de gestão de centro de dados. A série conta com uma plataforma de processador de alta velocidade, com duas portas Gigabit-Ethernet para redundância, opcional modem embutido e porta USB para modem (v.92) ou opções de 4G. Outros benefícios incluem um sensor de temperatura interna para monitoramento de nível de dispositivo e configuráveis ​​de pinos para portas seriais.
