EPU

 

Eficiência energética

O ASUS EPU (Energy Processing Unit - Unidade de Processamento de Energia) proporciona uma gestão completa da energia do sistema detectando as cargas atuais do PC e moderando inteligentemente a atribuição de energia aos componentes críticos em tempo real - ajudando a poupar energia e dinheiro!

 

 

Suporta USB 3.0

 

Desfrute de velocidades de transferência ultra-rápidas a 4.8Gbps com USB 3.0 - a norma de conetividade mais recente. Concebida para ligar facilmente componentes e periféricos da próxima geração, a norma USB 3.0 transfere dados 10x mais rápido e é retrocompatível com componentes USB 2.0

 

 

Turbo Key

 

ASUS Turbo Key permite o usuário tornar o botão para ligar o PC em uma botão de overclocking físico. Após a fácil configuração, Turbo Key pode aumentar a performance, sem interromper o trabalho ou jogos - com apenas um toque!

 

 

ASUS Proteção Anti-Surge

 

Anti-Surge Protege a placa-mãe de danos causados por picos de energia da unidade da fonte de alimentação (PSU). Proteção Anti-Surge detecta picos de energia e previne que eles se espalhem instantaneamente. Também cortar o fornecimento no caso de picos de energia para proteger o sistema.

1. Detecção instantânea de picos de energia.

2. Ativa proteção contra potenciais danos ao PC.

 

 

8 canais com Som de alta definição

 

Desfrute de áudio top de linha no seu PC! O CODEC integrado de 8 canais de áudio HD (Áudio de alta definição, nomeado anteriormente como Azalia) fornece uma saída de áudio de 192KHz/24-bit de alta qualidade, funções de jack-sensing, retasking e tecnologia multi-streaming que permite enviar streams de diferentes áudios para destinos diferentes. Pode agora falar com os seus parceiros através de headphones enquanto joga um game em rede. Tudo isto feito em um só computador

 

 

C.P.R. (CPU Parameter Recall)

 

O recurso C.P.R. BIOS restaura automaticamente as configurações da CPU quando o sistema trava por falha no overclocking. C.P.R. elimina a necessidade de abrir o gabinete para limpar os dados do RTC. Simplismente desligue e reinicie o PC, e a BIOS automaticamente será restaurada para a configuração padrão

 

 

EZ-Flash 2

 

Atualize a BIOS com um pendrive USB antes de instalar o Sistema Operacional

EZ Flash 2 é uma ferramenta amigável para atualização da BIOS. Os usuários podem facilmente navegar e atualizar a BIOS com uma pendrive USB, mesmo antes de instalar o sistema operacional. Com apenas alguns clicks o usuário pode atualizar a BIOS sem ter que preparar um diskete ou usar um programa pelo sistema operacional

 

 

CrashFree BIOS 3

 

Recupere os dados da BIOS a partir de uma unidade flash USB

A ASUS CrashFree BIOS 3 permite aos utilizadores restaurarem a os dados corrompidos da BIOS a partir de uma unidade USB que contenha a pasta da BIOS.
