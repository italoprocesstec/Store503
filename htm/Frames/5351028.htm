Apresentando ScanSnap S1300i





Desempenho em um tamanho compacto!

 

 

Visão Geral

Em quase metade do tamanho de uma folha de papel, o PC portátil e Mac compatível ScanSnap S1300i é o scanner ScanSnap dupla face multi-page menor para um botão facilidade de uso no balcão ou na estrada. Empilhar até 10 páginas no alimentador automático de documentos (ADF) e pressione o botão azul.

 

 

Faster frente e verso, multi-page de varredura

ScanSnap S1300i digitaliza ambos os lados de uma página em até 12 páginas por minuto directamente para PDF, PDF pesquisável, JPEG, ou para aplicações de escritório como o Word ou o Excel. O scanner portátil ScanSnap S1300i reconhece automaticamente o tamanho de cada documento, detecta e corrige a inclinação, e mostra imagens em sua orientação adequada. Você pode até optar por ter páginas em branco removidos automaticamente

 

 

Funciona com USB Bus Power por Portabilidade extra

ScanSnap S1300i vem com um adaptador AC e por conveniência móvel adicional, o ScanSnap S1300i pode ser alimentado através de portas USB do computador

 

 

Mais opções de digitalização = Produtividade do menu Mais Rápido

O Menu Rápido permite uma transição rápida entre diferentes opções de digitalização. Basta pressionar o botão de digitalização e escolher o que deseja ScanSnap fazer

Mostrado: Menu rápido para Windows

 

 

Digitalizar para ScanSnap Organizer: Gerir o seu PDF e JPEG

Digitalizar para CardMinder: digitalizar cartões de visita e extrair automaticamente informações de contato

Scan to Folder: scans loja PDF para uma pasta

Digitalizar para e-mail: Elimine fax e compartilhar documentos eletrônicos mais de e-mail

Digitalizar para Imprimir: Use ScanSnap com a impressora para criar cópias

Digitalizar para Imagem Pasta: scans loja JPEG para uma pasta

Digitalizar para Cloud: digitalizar para Evernote®, Google®Docs ™, Salesforce CRM, Salesforce Chatter, SugarSync e SharePoint (para Windows®)

Digitalizar para Word (.doc): digitalizar diretamente para um arquivo editável do Word

Digitalizar para Excel (.xls): digitalizar diretamente para um arquivo Excel editável

Digitalizar para PowerPoint (.ppt): digitalizar diretamente para um arquivo PowerPoint editável
