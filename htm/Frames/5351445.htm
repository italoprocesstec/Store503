Visão Geral

 

Servidor 2U em rack convencional oferece equilibrada desempenho e escalabilidade mid-range para uma ampla gama de cargas de trabalho comuns de negócios.

 

Projetado para centros de dados que precisam de um preço acessível, virtualização pronto, servidor em rack de dois soquetes, o PowerEdge R530 é uma excelente plataforma para uma ampla gama de aplicações comuns de negócios e de pequena escala virtualização. O R530 também é altamente atraente como um desenvolvimento dedicado ou servidor de backup.
