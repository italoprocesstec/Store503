O Switch KVM USB 2 portas DVI com kit de áudio permite-lhe controlar 2 computadores com um conjunto de controles de console com a tecnologia de alta definição de vídeo digital de imagem (DVI)

 

 

Visão Geral

 

Gerenciar 2 computadores que possuem portas DVI e USB com o toque de um botão com um conjunto de teclado USB, mouse USB, monitor DVI, microfone e alto-falantes. Desfrute de imagens de qualidade DVI de ambos os seus computadores. Não há necessidade de cabos adicionais, todos os cabos necessários estão incluídos. Ouvir música em um computador enquanto trabalha no outro. Com Hot-Plug, Auto-Scan e operação Hot-Key, este dispositivo versátil é a escolha perfeita para a operação de 2 computadores.

 

 

Características

 

·        2 portas switch KVM DVI USB do desktop compacto com áudio e microfone

·        Suporte ao console USB para o teclado USB, mouse e dispositivos USB de alta velocidade

·        Áudio e microfone comutação para uma experiência multimédia ininterrupta através de multi-plataformas.

·        USB 2.0 / 1.1 especificação de potência interface USB compatível

·        Comutação simultânea independente do PC e áudio e canal mic

·        Suporte multi-plataforma para PCs e Macintosh G3 / G4, iMac

·        Hot-plugging PCs para manutenção sem desligar o chaveador KVM ou PCs

·        Alternando cliente de software para fácil comutação software na plataforma Windows e Mac

·        Usuários hotkeys definível das teclas de atalho adicionais de troca de porta universal - ScrollLock + ScrollLock - para não-Windows OS na plataforma PC.

·        A selecção de canais por botão de pressão, atalhos de teclado e cliente de software de comutação (para Windows / Mac)

·        Energia interface USB Indicadores LED para fácil monitoração do status

·        Resolução de vídeo digital de 1920 x 1200

·        Garantia limitada de 2 anos
