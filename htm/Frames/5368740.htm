O Switch KVM USB 8 portas com On Screen Display (TK-804R) permite-lhe gerir vários computadores com apenas um conjunto de teclado, monitor e mouse.

 

 

Visão Geral

 

Este rack dispositivo de montagem suporta USB e interface de PS / 2 no lado do console e interface USB no lado do PC. Os usuários podem alternar entre PCs usando hot-chave, botão ou On Screen Display empurrar para a gestão flexível. PCs hot-plug rapidamente para evitar tempo de inatividade. Um máximo de 64 conexões de PC são suportados quando você cadeia TK-804R (s). A 8-Port KVM USB múltipla é a escolha perfeita para a operação de múltiplos PCs
