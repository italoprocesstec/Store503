O Switch 1-Port KVM sobre IP (TK-IP101) fornece gerenciamento de servidor remoto de custo eficaz para vários usuários, independentemente do status do servidor

 

 

Visão Geral

 

Eliminar a necessidade de estar fisicamente na sala do servidor para solucionar problemas, reiniciar e editar funções de pré-inicialização. Economizar recursos e gerenciar os recursos do servidor e do PC de todo canto ou ao redor do mundo. Fornece acesso altamente seguro para Windows e clientes baseados em Java. Com até 16 contas de usuários ativos do TK-IP101 suporta 4 logins simultâneos. Uma interface de usuário intuitiva Web fornece controles de funções de nível do BIOS para aplicativos baseados em Windows. Use o TK-IP101 com controlador de força 8-Outlet Remoto da TRENDnet (TK-RP08) para facilitar o ciclo de alimentação remoto e reiniciar

 

 

Características

 

·        Gerenciar switches e servidores KVM remotamente através da Internet

·        Centralizar o gerenciamento de vários servidores quando conectado a um switch KVM convencional

·        Economizar tempo e recursos, gerenciando recursos remotamente

·        Controle de acesso de vários usuários

·        Gerenciamento de servidores remotos a partir do bios subir de nível para aplicações de nível de gui

·        Acesso seguro ultra com autenticação pki 1024 bits completo e criptografia ssl de 256 bits

·        Desfrute de resolução 1600x1200 crisp

·        Interface de usuário intuitiva web permite upgrades e backups

·        Suporta sessões telnet amigável e acesso ftp

·        Use com tk-rp08 trendnet para controlar ligar / desligar o ciclo de alimentação de vários dispositivos remotamente

·        Funciona com ldap, radius e servidores do active directory

·        Conexões ppp serial ethernet rj-45 e apoiada

·        Conexão ppp sobre serial rs-232 com taxas de dados de até 1 mbps

·        Trabalha com clientes baseados em java windows® e para a compatibilidade cross-platform

·        Conexões de internet baixa largura de banda suportada com compressão jpeg
