A série DGS-1210 é a mais recente geração de D-Link Web inteligente Interruptores com D-Link Green tecnologia 3.0

 

 

Visão Geral

 

A série está em conformidade com o padrão IEEE 802.3az Energia padrão Ethernet Eficiente. Apoio para a gestão e configurações IPv6 garantir sua rede permanece protegida após a atualização de IPv4 para IPv61.

Ao oferecer gerenciamento de múltiplos opções, a série DGS-1210 permite rápida implantação, expansão de infra-estrutura, e sem costura atualizações de função. Segmentação pequenas e médias as empresas, os Switches Smart Web DGS-1210

Fornece funcionalidade, segurança e capacidade de gerenciamento para uma fração do custo padrão de propriedade. O novo DGS-1210-28P é um Switch PoE Inteligente Web que fornece 24 portas 10/100/1000 Base-T PoE

Com 4 portas Gigabit SFP. Além IEEE 802.3af cumprimento, portos 1-4 fornece até 30 watts de potência saída seguinte 802.3at IEEE. O projeto permite mais flexibilidade na alocação de potência para uma variedade de alimentado dispositivos com custos de instalação acessíveis.

 

 

Economia de enregia

 

Os Switches DGS-1210 são capazes de conservar energia, sem sacrificar o desempenho operacional ou funcionalidade usando D-Link Green tecnologia 3.0. Usando o padrão Ethernet Energia Eficiente, a rede irá automaticamente diminuir o consumo de energia quando o tráfego é baixo, sem nenhuma configuração necessária. Para ambientes não totalmente apoiam o padrão, DGS-1210 switches oferecem opções avançadas de economia de energia, incluindo as configurações de fechamento de porta e fechamento, espera LED, e hibernação sistema baseado em perfis de tempo personalizados. Os perfis de tempo também podem ser aplicados ao comutador PoE de modo que o consumo de energia desnecessário pode ser guardado fora do horário. A série DGS-1210 também pode detectar o comprimento dos cabos ligados para reduzir automaticamente o consumo de energia nas ligações de cabos curtos.

 

 

 

Gestão Fácil

 

A série D-Link Web Smart Switch é projetada para um gereciamento fácil. Todas as configurações podem ser feitas através de uma interface Web, independentemente do sistema do PC host operacional. Além disso, a interface do usuário da web contém 10 opções de idioma para tornar as operações mais simples. Na primeira instalação, o utilitário SmartConsole descobrirá automaticamente todos os switches da D-Link inteligentes na rede, permitindo que os administradores para atribuir endereços IP e máscaras de sub rapidamente. Ele também permite atualizações de firmware simultâneo para múltiplos switches, economizando uma grande quantidade de tempo. Comandos de gerenciamento importantes, tais como firmware download ou arquivo de configuração, oferecem um método sofisticado de operações em lote de múltipla.

 

 

VLAN e VOIP

 

A série DES-1210 suporta autovigilância VLAN (Auto Surveillance VLAN, ASV) e Auto Voice VLAN que são mais adequados para VoiP e implementações de vídeo vigilância. Autovigilância D-Link é uma nova tecnologia de ponta que a série D-Link Web Smart Switch proporciona. Esta tecnologia consolida transmições de dados e vídeo de segurança em um único switch. A ASV também garante a qualidade de vídeo em tempo real para o monitoramento e controle, sem comprometer a transmissão de dados da rede convencional. A tecnologia Auto Voice VLAN aumenta serviço de VOiP colocando automaticamente o tráfego de voz a partir de um telefone IP a um VLAN atribuído. Com maior prioridade e VLAN individual esses recursos garantem a qualidade e segurança ao tráfego VoIP. Além disso o DSCP faz os pacotes de ethernet de diferentes níveis de serviço a ser atribuído ao tráfego de rede. Como resultado, esses pacotes de voz e vídeo têm precedência sobre outros pacotes. Além disso, com o controle de largura de banda, os administradores de rede podem reservar largura de banda para funções importantes que requerem uma largura de banda maior, ou pode ter maior prioridade.

 

 

Características Layer 2

 

Equipado com uma linha completa de recursos L2, essas opções incluem Snooping IGMP, Port Mirroring, Spanning Tree e Link Aggregation Control Protocol (LACP).O função de controle de fluxo (IEEE 802.3x) permite aos servidores conectarem-se diretamente ao switch para uma transferência de dados rápida e confiável. A 2000 Mbps Full Duplex, as portas Gigabit fornecem streaming de dados de alta velocidade para servidores com perda mínima de transferência de dados. Recursos de manutenção de rede incluem Loopback Detecção e Diagnóstico de cabo. Detecção de Loopback é usado para detectar loops criado por uma porta específica e desligar automaticamente a porta afetada. O recurso de diagnóstico de cabo projetado principalmente para administradores de e representantes de serviço ao cliente, pode descobrir rapidamente o tipo de erro e determinar a qualidade do cabo.

 

 

Proteja a sua rede

 

A Engine Safeguard, da D-Link, inova ao protege os Switches contra enchentes tráfego causado por ataques de vírus. Os switches também suportam autenticação baseada em portas 802.1X, permitindo que aos clientes da rede a serem autenticados através de servidores RADIUS externos. Além disso, a Access Control List (ACL) recurso aprimora a segurança da rede e protege a rede por triagem tráfego de MAC ilegal ou endereços IP. Prevenção ARP Spoofing evita intrusos maliciosos de enviar mensagens falsas maciças ARP através de uma fonte manipulado. Isso protege os dados importantes de serem roubados, e evita desperdício de ciclos de CPU nestes pacotes. Para maior segurança, o servidor DHCP conta com triagem blocos desonestos no servidor DHCP pacotes de portas de usuários para evitar a atribuição de IP não autorizado.
