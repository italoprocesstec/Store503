Custo-benefício, Wireless-N Conectividade

 

 

 

Visão Geral

 

Aumentar a produtividade e reduzir os custos utilizando o Wireless-N dupla Radio Access Point Cisco WAP131 com PoE. Este nível de entrada, de classe empresarial, ponto de acesso à área de trabalho oferece conectividade 802.11n dual-rádio e rádios de banda dupla simultâneos. Ele também tem um over-Ethernet Power-única porta (PoE) Gigabit Ethernet.

 

O WAP131 é uma solução ideal, acessível para as pequenas empresas que necessitam de conectividade básica Wi-Fi e acomodar menos usuários

 

 

Características e Capacidades

 

A Cisco WAP131 oferece rádios de banda dupla simultâneas para melhorar a cobertura e capacidade do usuário. A porta Gigabit Ethernet permite que o AP para ser alimentado por switch PoE para ajudar a reduzir os custos de cabeamento e fiação. Inteligente de qualidade de serviço (QoS) permitem que você priorizar o tráfego sensível à largura de banda para voz sobre IP (VoIP) e aplicações de vídeo. A Cisco WAP131 é fácil de configurar e usar, com configuração baseada em assistente intuitivo para chegar até você e funcionando em minutos. Um projeto de desktop atraente, com um suporte integrado permite que você coloque o AP em sua mesa na vertical ou na horizontal.

 

Entre seus benefícios, este ponto de acesso:

 

Fornece conectividade 802.11n de baixo custo para clientes de 2,4 GHz e 5 GHz

Oferece quartos individuais POE Gigabit Ethernet para conexão a um switch PoE

Funciona direito fora da caixa com fácil instalação, configuração baseada em web simples, e um assistente

Oferece paz de espírito com uma garantia vitalícia limitada Hardware

 

 

Especificações gerais

 

Concurrent suporte de rádio de banda dupla para até 300 Mbps por rádio para otimizar a capacidade e cobertura

Uma única interface Gigabit Ethernet LAN para um uplink de alta velocidade para a rede com fio

Design de desktop com suporte integrado para fácil acesso e colocação

QoS inteligentes para priorizar o tráfego de rede

Modo Ponte grupo de trabalho para expandir a sua rede de conexão sem fios a uma segunda rede Ethernet
