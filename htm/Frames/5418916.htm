Projetado para utilização na Automação Comercial, ideal para uso de Check-outs

 

 

Visão Geral

 

Projetado para ocupar menos espaço no checkout, o Newera E3 PRO possui as dimensões L85mm X A270mm X P300mm. Possui placa MINI-ITX com processador Intel Celeron DUAL CORE integrado, cooler de exaustão para suportar ambientes agressivos, alta conectividade para interligação com periféricos de automação comercial e consome menos energia se comparado aos computadores convencionais.
