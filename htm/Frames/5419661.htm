Visão Geral

 

Servidor Intel® família Board é otimizado para desempenho de negócios

 

O documento destaca a família Intel® Server Board S1200v3RP, que fornece o desempenho do servidor confiável, segura para aplicações de pequenas empresas. A família de produtos inclui sistemas de rack e pedestal

 

 

Características

 

• Suporta o processador Intel® Xeon® E3-1200 v3

• Quatro LR-DIMMs em até 1600 MT por segundo

• dupla integrada Intel® Ethernet Controller

• Intel® Trusted Execution (Intel® TXT)

• conector mezzanine opcional

• Gerenciamento remoto opcional Intel® Módulo 4 (Intel® RMM4)
