DirectX® 12 é um novo ", consola-como" API gráfica da Microsoft que capacita os desenvolvedores de jogos com controle mais direto e óbvio de hardware PC.

 

 

Visão Geral

 

Este controle direto ou "explícito" melhor expõe os recursos da AMD Radeon ™ GPUs de hardware para produzir um maior rendimento hardware e, em última análise, mais desempenho para os usuários. Para colocá-lo simplesmente: hardware muito mais eficiente através de um software inteligente! A critério de um desenvolvedor de jogos, esta eficiência superior pode ser gasto em taxas de quadros mais altas, menor latência (VR), menor consumo de energia, uma melhor qualidade de imagem, ou algum equilíbrio calculada de todos os quatro. Em qualquer cenário, os jogadores podem se beneficiar muito da escolha de hardware AMD para executar seu jogo favorito DirectX® 12.

 

 

AMD Crossfire ™ 

Aproveite o poder dos múltiplos GPUs

A tecnologia AMD CrossFire ™ é a plataforma de games em desempenho multi-GPU. Desbloquear power-dominando jogo, AMD CrossFire ™ aproveita o poder de cartões de dois ou mais gráficos discretos trabalhando em paralelo para melhorar drasticamente jogos performance.1 tecnologia AMD CrossFire ™ pronto placas gráficas caber praticamente todos os orçamentos. Com a flexibilidade para combinar duas, três ou quatro GPUs, a tecnologia AMD CrossFire ™ é a solução perfeita para aqueles que exigem desempenho extremo.
