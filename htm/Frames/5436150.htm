O servidor ideal para pequenas empresas, o Windows Server 2012 R2 Essentials simplifica a integração de aplicativos e serviços baseados na nuvem opcionais, como o Microsoft Office 365 e Microsoft Azure.

 

 

Visão Geral

 

No coração da visão Microsoft nuvem OS, Windows Server 2012 R2 traz a experiência da Microsoft oferecer serviços em nuvem em escala global em sua infra-estrutura. Windows Server 2012 R2 oferece recursos interessantes e melhorias por meio da virtualização, armazenamento, rede, infra-estrutura de desktop virtual, acesso e proteção da informação, e muito mais
