A série de switches HPE 1405 é composta de switches de Camada 2, plug-and-play, não-gerenciados, com porta fixa, projetados para pequenas empresas.

 

 

Visão Geral

 

Esses switches são pré-configurados com conexões que usam cabos de cobre Ethernet de baixo custo.

Eles são certificados para classe B, energeticamente eficientes e silenciosos. Eles são compatíveis com recursos de administração práticos, como negociação automática.

Projetados para alta confiabilidade, longa duração e baixo consumo de energia, eles são ideais para empresas preocupadas com o custo total de propriedade

 

 

Fácil utilização

 

A série de switches HPE 1405 oferece simplicidade de plug-and-play sem gerenciamento.

A tela LED completa com indicadores por porta fornece uma visão imediata do status, das atividades, velocidade e operação full-duplex.

O controle de fluxo ajuda a garantir comunicações confiáveis durante a operação full-duplex.

 

 

Conectividade e desempenho

 

A série de switches HPE 1405 com Auto-MDIX se ajusta automaticamente para cabos straight-through ou crossover, em todas as portas 10/100 e 10/100/1000.

O suporte a pacote Jumbo para portas Gigabit Ethernet oferece apenas o tamanho de quadro até 9216 bytes, para aumentar o desempenho de grandes transferências de dados.

O recurso de negociação automática half-/full-duplex em cada porta dobra a taxa de transferência de cada porta.

 

 

Qualidade de Serviço (QoS)

 

A série de switches HPE 1405 tem priorização IEEE 802.1p, oferecendo dados para dispositivos com base na prioridade e no tipo de tráfego.

Os switches suportam Ethernet energeticamente eficiente.
