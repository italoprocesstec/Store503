Modernize sua rede

 

 

 

Visão Geral

 

N-Series é uma família de eficiência energética e de baixo custo 1GbE e 10GbE interruptores projetados para a modernização e infra-estrutura de rede de escala. A Dell Networking Series N3000 oferece alta disponibilidade GbE muda para Layer 3 acesso ou agregação, proporcionando fácil de gerenciar soluções que:

 

·      Use MLAG para multipath redundância sem loop sem árvore estendida para permitir a plena utilização de largura de banda e alta disponibilidade

·      Promover uma maior interoperabilidade através de interfaces com o Rapid Per VLAN Spanning Tree da Cisco (RPVST +) 1 e os dispositivos que utilizam o Cisco Discovery Protocol (CDP)

·      Oferecer uma escolha mais flexível na sua rede, unindo produtos com os mais recentes protocolos de padrão aberto

·      Incluem recursos avançados de 3 de roteamento, segurança e escalabilidade IPv4 e IPv6 Camada

·      Plug-and-play configuração com matrizes de armazenamento Dell EqualLogic iSCSI com a configuração iSCSI de um comando que alivia a configuração de várias etapas e erros de configuração potenciais

·      Incluir OpenFlow 1.3 suporte para interoperabilidade com os controladores OpenFlow padrão da indústria.

 

 

Fornecer energia limpa para a rede dispositivos com PoE + apoiar

A criação de redes Dell Série N3000 tem opções para até 48 portas de Power-over-Ethernet Plus (PoE +) sem uma fonte de alimentação externa. Isso permite que você conectar dispositivos com requisitos de alta potência (até 30,8 watts) sem executar um cabo de alimentação separado. PoE + é particularmente útil em edifícios mais antigos, onde a instalação de energia para vários locais, difundidas pode ser proibitivamente caro.

 

 

Projetado para a eficiência

 

De construção para ajudar a manter os custos operacionais baixos, a série N3000 pode operar em temperaturas de até 113 ° F (45 ° C) para ajudar a reduzir os custos de resfriamento. Funcionalidades concebidas para melhorar a eficiência geral incluem:

 

·      Eficiente de energia de alimentação com certificação 80PLUS hot-swap com opção de abastecimento de dupla energia

·      Capacidade de energia suficiente para fornecer até 48 portas de poe + (até 30,8 watts) em um espaço de 1U

·      Ventiladores redundantes de refrigeração de velocidade variável

·      Ethernet e de baixo consumo phys energeticamente eficiente para ajudar a reduzir o poder para inativo portos e das ligações ociosas

 

 

Conjunto de recursos avançados para simplificar o gerenciamento

A Dell Networking Series N3000 é concebido para facilitar a implantação, a interoperabilidade excelente e uma curva de aprendizagem baixa para administradores de rede. A única versão OS (Rede Dell OS 6) permite-lhe manter as configurações consistentes em todos os produtos N-Series com um conjunto abrangente de recursos que inclui:

 

·      Um comando comum Line Interface (CLI) e interface gráfica do usuário (GUI) que usam uma linguagem de comando bem conhecido para ajudar os administradores de rede qualificados se tornar produtivo rapidamente

·      auto-configuração USB que ajuda os administradores de rede implantar rapidamente configurações espelhadas para inúmeros dispositivos, simplesmente inserindo uma chave USB

·      Uma variedade de empresa de classe apresenta com comandos de configuração e gerenciamento familiares e intuitivas

·      Os últimos protocolos de padrão aberto e tecnologias de integração inteligentes para ajudar a tornar as redes de vários fornecedores que operam sem problemas possível

 

 

Implante com confiança em qualquer escala

Com alto rendimento e capacidade de lidar com cargas de trabalho inesperados, os interruptores N3000 Series são Camada 3 soluções ideais para crescimento rápido redes GbE que requerem a agregação de alta densidade com redundância sem costura e disponibilidade.

 

·      As taxas de dados de até 260Gbps (full-duplex) e uma taxa de transmissão de até 193Mpps

·      Módulo de expansão hot swap suporta dual SFP + e dual 10GBaseT

·      Built-in portas de empilhamento e cabos de empilhamento N-série padrão suportar uma arquitetura de empilhamento de até 12 unidades 84Gbps

·      Interruptor acumula até 624 portas 1GbE pode ser gerido a partir de uma única tela ou endereço IP
