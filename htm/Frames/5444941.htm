Netgear Gsm7252Pl licença Layer 3 upgrade para desbloquear IPv4 / IPv6 capacidades de roteamento dinâmico de GSM7252PS Gigabit Switch

 

 

Visão Geral

 

Projetado para redes corporativas e os níveis de core / agregação de redes de pequenas e médias empresas, L3 atualizações fornecer protocolos de roteamento avançados como OSPF, VRRP e multicast para aplicações convergentes

 

 

Principais características

 

·      Licença (licença de actualização)

·      Para ProSafe GSM7252PS

 

 

Atualização GSM7252PL licença desbloqueia seguintes características:

·      IPv4 / IPv6 RIP, OSPF, VRRP

·      Roteamento IPv4 / IPv6 Multicast

·      IPv6 L2, L3, L4 ACL e QoS

·      GSM7328S-200 funcionalidade equivalente elevação set

 

 Conteúdo do pacote

Não se aplica, GSM7252PL é uma licença eletrônica
