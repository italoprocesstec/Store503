Os Switches Cisco Catalyst 3850 são a próxima geração de empresa de classe, empilháveis, os switches da camada de acesso.

 

 

Visão Geral

 

Eles fornecem convergência total entre as redes com e sem fios em uma única plataforma. Esta convergência é construída sobre a resiliência dos novos 480 Gbps tecnologias, Cisco StackWise e Cisco StackPower.

 

A tabela a seguir apresenta comparações de alto nível dos diferentes modelos disponíveis no Cisco Catalyst 3850 Switch Series família. Você pode ver todos os modelos com:

 

LAN software da base

Software Base de dados de IP

Software Serviços IP

Os clientes precisam para comprar o modelo de software Base de dados de IP ou o modelo de software Serviços de IP para usar um switch Cisco Catalyst 3850 como um controlador sem fios. Modelos LAN Base de exigir a compra de um cabo StackPower separadamente para ativar a funcionalidade StackPower
