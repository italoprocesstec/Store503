Suporte para processadores da 6ª Geração Intel Core- Dual Channel DDR3/DDR3L - 4 DIMMs

 

 

Visão Geral

 

Suporte Gráfico para Crossfire - PCIe Gen3 x4 M.2 Connectors com taxa de transferência de até 32Gb/s (suporte a PCIe e SATA SSD)- SATA Express com velocidade de transferência de dados de até 16Gb/s- UEFI DualBIOS da GIGABYTE

 

 

Recursos

 

·      Suporte aos processadores de 6ª Geração Intel® Core™

·      Dual Channel DDR3/ DDR3L, 4 DIMMs

·      Suporte Gráfico a 2-Way com Premium PCIe

·      PCIe de 3ª Geração x4 M.2 com taxa de transferência de até 32Gb/s (suporte a PCIe NVMe & SATA SSD)

·      Conector SATA Express com velocidade de transferência de até 16Gb/s

·      8 canais de áudio HD com capacitores de áudio de alta qualidade

Chipset

Intel® B150 Express Chipset

Memória

4 x DDR3 DIMM sockets supporting up to 64 GB of system memory

* Due to a Windows 32-bit operating system limitation, when more than 4 GB of physical memory is installed, the actual memory size displayed will be less than the size of the physical memory installed.

Dual channel memory architecture

Support for DDR3/DDR3L 1866(OC)/1600/1333 MHz memory modules

Support for ECC UDIMM 1Rx8/2Rx8 memory modules (operate in non-ECC mode)

Support for non-ECC UDIMM 1Rx8/2Rx8 memory modules

(Please refer "Memory Support List" for more information.)
