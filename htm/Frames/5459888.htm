Flexibilidade

 

 

 

Ampla gama de diferentes entradas e saídas

Várias entradas e saídas para a distribuição de 120 V, 208 V ou 230 V de energia para vários terminais. Oferece aos usuários a capacidade de se adaptar a diferentes requisitos de energia. Os dispositivos suportam até 14.4kW ramo chicote.

 

 

Disponibilidade

 

 

 

Recursos de gerenciamento remoto

 

Interfaces de gerenciamento de rede com recursos completos que assegurem a gestão baseada em padrões através da Web, SNMP e Telnet. Permite aos usuários acessar, configurar e gerenciar unidades a partir de locais remotos para poupar tempo valioso. (Nota: apenas para sistemas que possuem uma interface de rede)

 

 

Poder com 2 entradas

 

Através de configuração redundante os dispositivos conectados via duas linhas são energizados. Se a linha principal, a unidade muda automaticamente para a linha secundária.

Benefícios para o seu negócio

 

 

 

Para cremalheira

 

Ocupa apenas 2U espaços de rack valiosos. Coloca o poder disponível lá onde eles são mais necessários - no rack, perto do equipamento

 

 

Exibição de informações

 

Exibe o status da AC tensão de alimentação: a fonte, que é definida como cuidados primários, e a fonte, que atualmente abastece as unidades. Caracterizado uma representação gráfica do fluxo de corrente a partir da fonte é fornecido para o sistema. Os usuários podem com apenas um olhar para verificar o fluxo de corrente e o estado do fornecimento, sem conectar dispositivos adicionais. (Nota: As unidades não baseados em rede só fornecer uma indicação de estado e não é uma representação gráfica do fluxo de corrente)

 

 

Poder com 2 entradas

 

Através de configuração redundante os dispositivos conectados via duas linhas são energizados. Se a linha principal, a unidade muda automaticamente para a linha secundária.
