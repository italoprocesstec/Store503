Nobreak NHS Senoidal Premium PDV GII Bivolt 1000VA - 91.A0.010001



Visão Geral


A Linha de Nobreaks Senoidal utiliza a tecnologia DSP (Digital Signal Processor) que agrega alta performance, apresentando como principal característica a forma de onda senoidal com distorção menor que 1% em modo inversor, ideal para sistemas eletroeletrônicos sensíveis que exigem ondas senoidais.

Entre outras vantagens, os Nobreaks da Linha Senoidal realizam medidas TRUE RMS, possuem gerenciamento das baterias, visualização precisa dos parâmetros de gerenciamento de energia através de monitoramento via interface RS232 com software proprietário e gratuito.
